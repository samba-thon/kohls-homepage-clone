# Kohl's Homepage Clone

A responsive HTML and CSS recreation of the Kohl's department store homepage, built as a bootcamp assignment.

## Project Overview

This project is an educational clone of the Kohl's homepage, designed to demonstrate proficiency in semantic HTML5 and CSS styling. The clone includes the key sections and layout patterns found on the live Kohl's website.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Grid, Flexbox, media queries
- **Font Awesome** - Icon library for UI elements

## Notes

- This is an educational project using placeholder branding ("khol's")
- Product images are sourced from Kohl's media server
- All styling is custom CSS without external frameworks
- No JavaScript functionality implemented
